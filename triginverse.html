Review of Inverse Trigonometric Functions

1. Definition of Inverse Trigonometric Functions

The inverse trigonometric functions are the inverses of the standard trigonometric functions, restricted to certain domains to ensure they are one-to-one. They are:

  • Arcsine: y = arcsin(x) means sin(y) = x, where -π/2 ≤ y ≤ π/2.
  • Arccosine: y = arccos(x) means cos(y) = x, where 0 ≤ y ≤ π.
  • Arctangent: y = arctan(x) means tan(y) = x, where -π/2 < y < π/2.
  • Arccotangent: y = arccot(x) means cot(y) = x, where 0 < y < π.
  • Arcsecant: y = arcsec(x) means sec(y) = x, where 0 ≤ y < π/2 or π ≤ y < 3π/2.
  • Arccosecant: y = arccsc(x) means csc(y) = x, where -π/2 ≤ y ≤ π/2, excluding 0.

2. Properties of Inverse Trigonometric Functions

Reciprocal Identities:

arcsin(x) + arccos(x) = π/2, arctan(x) + arccot(x) = π/2

Derivatives:

d/dx [arcsin(x)] = 1/√(1 - x²), |x| < 1
d/dx [arccos(x)] = -1/√(1 - x²), |x| < 1
d/dx [arctan(x)] = 1/(1 + x²)
d/dx [arccot(x)] = -1/(1 + x²)
d/dx [arcsec(x)] = 1/|x|√(x² - 1), |x| > 1
d/dx [arccsc(x)] = -1/|x|√(x² - 1), |x| > 1

3. Example Problems

Example 1: Evaluating Inverse Trigonometric Functions

Find arcsin(1/2).

Solution: arcsin(1/2) = π/6

Example 2: Differentiating an Expression

Find d/dx [arctan(2x)].

Solution: 2/(1 + 4x²)

Example 3: Evaluating a Definite Integral

Evaluate ∫ dx/(1 + x²) from 0 to 1.

Solution: π/4

4. Practice Problems and Answers

  • Compute arccos(-√3/2). Answer: 5π/6.
  • Find d/dx [arcsin(3x)]. Answer: 3/√(1 - 9x²).
  • Evaluate ∫ dx/√(1 - 4x²). Answer: (1/2) arcsin(2x) + C.
  • Find arctan(tan(3π/4)). Answer: -π/4.
  • Compute ∫ dx/(x² + 4). Answer: (1/2) arctan(x/2) + C.

This review provides a solid foundation for understanding inverse trigonometric functions, their properties, and applications in calculus.
